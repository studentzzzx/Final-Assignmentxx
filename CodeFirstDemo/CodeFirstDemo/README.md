# Entity Framework Code-First Demo

## Overview
This project demonstrates the **Code-First** approach in Entity Framework.  
It creates a simple `Student` database and inserts one student record.

## Creator
- **Mohamad Ahamad**

## Steps
1. Define a `Student` class (POCO).
2. Create a `SchoolContext` class inheriting from `DbContext`.
3. Add a student named *Mohamad Ahamad* and save changes.

## Requirements
- Visual Studio
- .NET Framework
- EntityFramework NuGet package

## Run Instructions
1. Clone the repo.
2. Open in Visual Studio.
3. Run the console app.
4. Check the generated `SchoolDB` database.
